CIC-UNSW-NB15 Research Dataset

Research Foundation

The CIC-UNSW-NB15 represents a state-of-the-art network security dataset, meticulously engineered through the augmentation of the original UNSW-NB15 dataset using advanced CICFlowMeter feature extraction techniques. This comprehensive dataset captures 100GB of real-world network traffic over a 48-hour period using the IXIA PerfectStorm traffic generation platform.

Advanced Feature Engineering

The dataset incorporates 47 sophisticated features extracted through Argus and Bro-IDS analytical frameworks, systematically categorized into four primary domains:

Basic Features: Core network flow characteristics including packet counts, byte transfers, and connection states

Content Features: Deep packet inspection metrics analyzing payload characteristics and content patterns

Time Features: Temporal flow analysis including inter-arrival times, duration metrics, and timing correlations

Generated Features: Advanced computed metrics derived from statistical analysis and feature transformation

Statistical Balancing Methodology

To ensure realistic representation of production network environments, the dataset maintains an optimal 80:20 ratio between benign and malicious traffic samples. This carefully engineered balance preserves all malicious flow instances while implementing intelligent random sampling of benign traffic to achieve statistical significance.

Model Feature Selection

Optimized Feature Set: The Random Forest classifier utilizes 33 statistically significant features identified through rigorous feature importance analysis, including ACK Flag Count, Active statistics, Packet Size metrics, IAT measurements, Flow rates, and specialized network behavioral indicators.
